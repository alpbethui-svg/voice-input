● 阶段结束 handoff：输入助手 Windows 安装包

  1. 当前目标

  把 Windows 本地工具从“语音输入小工具”整理成长期可用的 输入助手，功能包括：

  - Alt+3：按住说话，Whisper 本地转写，本地 Ollama 小模型保守纠错，然后输入到当前光标。
  - Alt+4 / Ctrl+Alt+O：区域截图 OCR，识别文字复制到剪贴板。
  - 做成正式 Windows 安装包：
    - 有安装说明；
    - 有开始菜单快捷方式；
    - 可选开机自启；
    - 不把 Whisper / Ollama / 大模型 / torch / CUDA 大包塞进安装包；
    - 模型改为安装后按需准备或首次使用时下载。

  8. 剩余风险 / 注意事项

  8.1 dist 仍有 375 MB

  虽然已经从 4.3GB 降到 375MB，但仍包含一些 native 运行依赖，例如：

  cv2
  onnxruntime
  ctranslate2
  av
  numpy/openblas

  这些不是模型缓存，也不是 torch 大包。
  后续如果要继续瘦身，可以单独优化 RapidOCR / OpenCV / av 依赖，但不要在当前阶段贸然删 DLL，容易破坏 OCR 或 Whisper。

  8.2 Inno Setup 语言

  本机安装的 Inno Setup 没有 ChineseSimplified.isl，所以安装器目前用默认英文安装界面，但 InfoBeforeFile=说明.txt 是中文说明。

  如果后续要中文安装界面，需要额外放一个可用的中文 .isl 语言文件，或安装带中文语言文件的 Inno Setup 配置。

  8.3 日志位置

  打包后日志目前会落在 exe 运行目录的 _internal 附近。
  可用但不够理想。后续长期产品化时，可以改为用户目录：

  %LOCALAPPDATA%\输入助手\input_helper.log

  这不是当前阶段必须项。

  8.4 Whisper GPU 再确认

  源码模式之前确认过 GPU 可用。
  但打包后由于排除了 torch，GPU 检测改为由 faster-whisper / CTranslate2 自己尝试 CUDA。
  后续用户实际跑一次 Alt+3 后，需要看日志确认：

  加载模型 'medium' (device=cuda, int8_float16)
  模型加载完成 (device=cuda)

  如果看到 CUDA 失败回退 CPU，再针对 CTranslate2/CUDA DLL 处理。
